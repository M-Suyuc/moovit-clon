# Como incializar un proyecto de TypeScrit y corre un archivo

## 1 instalar ts-node

-npm install -D ts-node
-pnpm install -D ts-node

## 2 hacer un script en el package.json

"dev": "ts-node main.ts"

## incializar un proyecto de TypeScript en la consola

npx tsc --init

## correr el comando para ejcutar el archivo

pnpm run dev
